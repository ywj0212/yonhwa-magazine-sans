SIL Open Font License (OFL) Usage Notes

Yonhwa Magazine Sans is distributed under the SIL Open Font License,
Version 1.1 (OFL-1.1). This file is a short, practical checklist.

- Keep the full license text (`LICENSE`) in any distribution.
- If you modify the font, use a new primary font name and do not use
  upstream Reserved Font Names.
- Keep `THIRD_PARTY_NOTICES` with redistributed packages.
- You may bundle or sell the font with software, but not by itself.
- Documents created using the font are not covered by the OFL.

This note is not legal advice.
